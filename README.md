# AI Job Search System
### Built by Jason Darrow | jasondarrow.com

An AI-powered job search assistant that finds relevant 
job postings, customizes my resume and cover letter for 
each role, and tracks my application progress from 
submission to offer.

Built as a learning project to develop hands-on AI 
engineering skills while solving a real personal problem.

## What it does
- Scrapes job boards and filters postings by role and criteria
- Uses AI to tailor my resume and cover letter per job
- Tracks application status through the full pipeline
- Publishes progress publicly at jasondarrow.com
