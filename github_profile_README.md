## What I'm Building

### AI Job Search System
A Python pipeline that scrapes job listings from live APIs, enriches each company with real intelligence, and uses the Claude API to tailor my resume to specific postings.  It outputs a submission-ready Word document.  I built it because I needed it — and because building it proves the point better than talking about it.

**Stack:** Python · Claude API · Adzuna API · NewsAPI · DuckDuckGo · python-docx · pandas · openpyxl · AWS S3 · Route 53
